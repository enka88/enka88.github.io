Mailmerge Menggunakan Inkscape

Pada catatan kali ini, saya akan membagikan tutorial bagaimana membuat mailmerge menggunakan Inkscape. Metode ini sangat cocok diterapkan dalam pembuatan sertifikat, kartu nama, dan lain-lain. Apa saja perkakas yang diperlukan?

Software yang diperlukan

  • Inkscape
  • Ink-generator

Memasang Extension Inkscape

  • Untuk pengguna Windows, letakkan file generator.inx dan generator.py ke dalam folder C:Program filesInkscapeshareextensions (untuk semua User) atau C:UsersApplication DataRoaminginkscapeshareextension (untuk satu User)
  • Untuk pengguna GNU/Linux, letakkan file generator.inx dan generator.py ke dalam folder /usr/share/inkscape/extensions (untuk semua User) atau /home//.config/inkscape/extensions/ (untuk satu User)

Mulai Membuat “Mailmerge”

  • Siapkan desain kartu / sertifikat / lainnya yang hendak dibuat “mailmerge”-nya

  • Siapkan juga tabel data yang digunakan untuk mailmerge. Untuk nama kolom usahakan jangan ada Spasi, agar tidak menimbulkan Error.

    contoh tabel data untuk mailmerge di inkscape

    Simpan ke dalam format csv.

  • Jika mail merge yang dikehendaki membutuhkan gambar / foto. Simpan gambar-gambar / foto-foto tersebut ke dalam satu folder yang sama

  • Buatlah teks baru untuk variabel dengan format %VAR_%. Sesuaikan dengan nama kolom di file data tadi. Misal untuk kolom nama, kita buat teks %VAR_nama%

  • Jika menghendaki gambar, Import gambar dengan Import type Link

  • Untuk menggunakan Ink-Generator, klik menu Extensions > Export > Generator
